# FiCitas Web

Esta versión reemplaza la interfaz de escritorio por una aplicación web sencilla, moderna y funcional.

## Cómo usarla

1. Abre el archivo index.html en el navegador, o
2. Ejecuta un servidor estático desde la carpeta del proyecto:

```bash
python -m http.server 8000
```

Luego entra a http://127.0.0.1:8000/
